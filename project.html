MY PROJECTS



1.Smart Inventory Management System (SIMS)



  • Created a real-time web dashboard using PHP, Python, and MySQL.
  • Integrated sensors and microcontrollers to detect product movement and send data to database.
  • Aimed at optimizing stock management and predicting consumption trends.
  • Can be integrated with stock related websites. Machine learning to process the data.
  • Tech Used: PHP, Python, MySQL, Microcontrollers

2. Power Brick (Bio-Brick)

  • Biobrick made using industrial and agricultural waste.
  • Main components include high density polyethylene plastic, flyash and sand.
  • Aimed at reducting waste and promoting green materials and sustainability.
  • Prototype in progress. Insights and testing required.
  • Tech/Fields Used: Material Science, Waste Management
